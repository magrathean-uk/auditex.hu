What it does

  • Collects tenant audit evidence for authorised operators.
  • Records proof tables, API inventory, checksums, and blockers.
  • Keeps source, docs, issues, and releases on GitHub.

What it avoids

  • No mailbox or file body reads on the public audit surface.
  • No production writes during evidence collection.
  • No claims that the output replaces human review.

Run path

  1. CloneUse the GitHub repository as the source of truth.
