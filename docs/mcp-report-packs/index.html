MCP and report packs

Local MCP sits on top of saved evidence, not hype.

The site should explain that MCP is there to reason from saved bundle artifacts. It is not permission to invent unsupported conclusions or move raw secrets into third-party AI tools.

MCP inputs

Summaries, audit plans, API inventory, proof tables, validation output, and AI-safe context files from the local bundle.

What stays out

Raw credentials, secrets, raw tenant exports, and unnecessary personal data should not be pushed into third-party AI services.

Report packs

Handoff packs remain review material for operators and customers. They do not replace independent judgement.
